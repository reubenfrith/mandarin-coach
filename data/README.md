# data/

This directory contains all RAG source data loaded into ChromaDB at startup.

## Files

| File | Collection | Content | Count |
|---|---|---|---|
| `hsk_vocab.json` | `hsk_vocabulary` | Full HSK 1–6 word list — character, pinyin, English meaning, HSK level, part of speech | **4,991** |
| `grammar_rules.json` | `grammar_rules` | Curated Mandarin grammar rules — rule name, explanation, common English-speaker mistakes, correct/incorrect examples | **98** |
| `grammar_patterns.json` | `grammar_patterns` | Chinese Grammar Wiki points — name, explanation, structure(s), worked example, source URL | **217** |
| `error_patterns.json` | `error_patterns` | English-speaker error archetypes — error type, why it happens, correct usage, examples | **16** |
| `load_data.py` | All | Embeds + loads the reference files into ChromaDB (batched); run with `--force` to rebuild | — |
| `build_hsk_vocab.py` | — | Regenerates `hsk_vocab.json` from the upstream complete-hsk-vocabulary dataset (see below) | — |

## Sources & attribution

- **`hsk_vocab.json`** is generated by `build_hsk_vocab.py` from **[complete-hsk-vocabulary](https://github.com/drkameleon/complete-hsk-vocabulary)** by **Yanis Zafirópulos (drkameleon)** — **MIT License**, © 2026 Yanis Zafirópulos. We take the *old HSK 1–6* standard (~5,000 words) and map it to our schema; the 22 hand-curated example sentences from the original seed are preserved by word match. English glosses in that dataset derive from **[CC-CEDICT](https://www.mdbg.net/chinese/dictionary)** (CC BY-SA 4.0).
- **`grammar_patterns.json`** is ingested (via a one-off extraction) from **[chanind/cn-grammar-matcher](https://github.com/chanind/cn-grammar-matcher)** — the matcher *code* is **MIT-licensed**. The grammar-point **explanations and example sentences are © AllSet Learning, [Chinese Grammar Wiki](https://resources.allsetlearning.com/chinese/grammar/), licensed CC BY-NC-SA** — used here for a **non-commercial, educational** project with attribution. Each record keeps its `source_url` back to the Chinese Grammar Wiki. No HSK level is present in the source, so `hsk_level` is stored as `0` (unknown) and is **never displayed as a level**.
- **`grammar_rules.json`** and **`error_patterns.json`** are hand-curated for this project (grammar rules informed by the Chinese Grammar Wiki and standard textbooks; error patterns from L1-transfer research, some synthetic examples via Claude).
- The runtime **`dictionary_lookup`** tool also queries **CC-CEDICT** (mdbg.net) live and derives pinyin with **pypinyin**.

## Notes

- `personal_errors` collection is NOT pre-loaded — it is empty per user and grows as they interact with the app.
- Reference collections are re-seeded only when empty; use `load_reference_data(force=True)` (or `python data/load_data.py --force`) to rebuild after updating a source file.
- Collections are user-namespaced: `{user_id}_personal_errors`.
- The Task 6 embedding sweep (OpenAI vs BGE-M3, + BM25 hybrid) runs against these same collections.
